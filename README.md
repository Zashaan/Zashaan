# GoodWorks

Static front-end MVP for the GoodWorks community app.

## Local

Open `index.html` in a browser.

## Deploy

This project is ready for static hosting.

### Vercel

- Import the project folder or repo
- Framework preset: `Other`
- Output directory: `.`

### Netlify

- Import the project folder or repo
- Publish directory: `.`

### GitHub Pages

- Push the files to a GitHub repo
- Enable Pages from the repo settings
- Serve from the root or `main` branch
